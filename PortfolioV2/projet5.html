Détails du Projet

Mon stage s'est déroulé à Grenoble dans l’entreprise Flex For You. Je disposais d’une période de 10 semaines afin d’accomplir plusieurs missions qui ont pu évoluer au fur et à mesure du stage. La première mission, qui est celle pour laquelle j’ai été employé, était de générer un tableau en respectant les demandes d’un comptable travaillant pour l’entreprise Charignon. Ensuite j’ai dû réaliser l’importation de données depuis un fichier Excel vers XRP FLEX dans le contexte de l’entreprise TemaHome. Cela comprenait donc d’adapter le web service qui reçoit les données et de créer la macro qui envoie les données. Ce projet a été suivi d’un autre reprenant les mêmes bases mais sur une API différente. Enfin j’ai effectué un travail de recherche sur les meilleurs API d’IA pour compléter la description d’un article. Ce stage m'a permis d'acquérir de nouvelles compétences, de mieux comprendre les besoins des clients et de travailler en collaboration avec eux. J'ai également découvert l'importance de la maintenabilité du code et de la penser à long terme dans le développement de logiciels. Cette expérience m'a préparé aux complexités de la vie professionnelle et m'a appris à m'adapter à différents langages et outils informatiques.

Technologies Utilisées

  • VBA
  • C#
  • API
